# 🚁 에어로로직 성남 — 성남시 드론·로봇 융합 배송 거점 최적 입지 분석

> **수요·제약·유휴공간 다중 레이어를 결합해 사용자가 시나리오를 직접 선택할 수 있는 의사결정형 시각화**

---

## 📌 프로젝트 개요

성남시는 전국 최초 6년 연속 드론 실증도시로, K-드론배송 브랜드화를 추진하고 있습니다.  
그러나 **서울공항 관제권(시 면적의 80% 이상 영향)**, **고층 건물과 구릉지 지형**, **로봇 단독 배송의 지형적 한계**라는 복합적인 제약이 존재합니다.

이 프로젝트는 다음 핵심 질문에 답합니다:

- **Q1.** 법적·물리적 제약을 회피하면서, 수요가 충분하고 공공 유휴공간을 활용할 수 있는 거점은 어디인가?
- **Q2.** 제약 조건을 켜고 끄면 추천 거점·경로는 어떻게 달라지는가?
- **Q3.** 드론·로봇이 오토바이 대비 시간·탄소·접근성 면에서 얼마나 이득인가?

"가장 좋은 한 곳"을 추천하는 대신, **사용자가 제약 조건을 On/Off 하며 거점·경로를 직접 비교**하는 시뮬레이션형 의사결정 도구를 제공합니다.

## 📊 분석 파이프라인

총 5단계로 H3 격자 1,947셀에 걸쳐 분석을 수행합니다.

```
Step 1: 전처리·격자화
  행정동 경계 클리핑 → H3 격자 1,947셀 생성 → 건물 높이·DEM·도로망 정합

Step 2: 수요 레이어 산출
  카드10(카드소비) + 통신22(유동인구) + 기업3(법인현황) + 신용3(전입통계)
  → B2C·B2B·생활 수요를 0~1 정규화 → Ds 종합수요 지수

Step 3: 제약 레이어 5종 점수화
  공역(관제권) · 장애물(건물높이) · 소음 · 지형 경사 · 로봇 접근성
  → 셀별 제약 점수 산정

Step 4: 거점 선정 및 시나리오 생성
  공영주차장 171개 후보 + 시나리오 64종 조합 + Greedy 알고리즘 거점 선정

Step 5: 경로·KPI 산출
  거점별 배송권역·경로 48개 산출
  드론 vs 오토바이 시간·CO₂·ESG 지수 비교
```

## 🗺️ 대시보드 주요 기능

- **KPI 7종 실시간 갱신** — 거점 수 · 경로 수 · 시간 절감 · CO₂ 절감 · 기상 가용일 · 드론우세 % · ESG 지수

- **시나리오 64종 선택 패널** — 제약 조건 5종 체크박스로 조합 → 지도·KPI·차트 즉시 갱신

- **서버 불필요** — 단일 HTML 파일, 사전 계산된 JSON 데이터 내장

**기술 스택:** Leaflet · Plotly · Vanilla JS · Python (GeoPandas, H3, Shapely) · GeoPackage · Parquet
